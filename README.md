# NumericalMethods
A comprehensive collection of numerical analysis algorithms and mathematical models implemented in **C#**. This repository contains practical works and projects focused on computational mathematics and data processing, completed during my studies at Igor Sikorsky KPI.

## 🛠 Tech Stack & Tools
*   **Language:** C# (Core)
*   **Libraries:** MathNet.Numerics
*   **Key Concepts:** Object-Oriented Programming (OOP), Data Preprocessing, Algorithm Design

## Key Implemented Algorithms
This repository demonstrates the practical application of computational algorithms to solve complex systems and process data:
*   **Solving Systems of Linear Equations:** Implementation of exact methods, including LU-decomposition.
*   **Iterative Methods:** Implementation of Jacobi and Gauss-Seidel algorithms for large-scale matrices.
*   **Data Processing:** Algorithms for parsing, extracting, and structuring data from raw text files.

## AI-Assisted Development
Large Language Models (Claude, Gemini) were actively utilized during development for:
*   Refactoring and optimizing C# code architecture.
*   Debugging complex matrix operations within the MathNet library.
*   Generating edge-case scenarios for rigorous algorithm validation.
